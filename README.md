# Crash-Game-analysis of bcgames-
---

## 📊 **Анализ распределения коэффициентов в игре Crash**

**Цель проекта:**  
Исследовать статистические закономерности коэффициентов (множителей) в игре Crash на основе исторических данных.

**Зачем это нужно:**  
Понять, насколько распределение коэффициентов соответствует заявленной математической модели (обычно краш-игры имеют равномерное распределение или распределение с заданной вероятностью просадки). Выявить аномалии, проверить гипотезу о честности игры.

---

### 📁 **Данные**

- Файл с данными: `50k+.txt`
- Содержит **более 50 000** исторических значений коэффициентов (множителей), на которых произошел "краш".
- Данные представлены в виде списка чисел с плавающей точкой (например, `1.8`, `3.98`, `57.08`, `420.88` и т.д.).

---

### 🛠 **Ход работы (этапы анализа)**

#### 1. Загрузка и предварительная обработка данных
- Считали сырые данные из файла.
- Удалили артефакты и "шумы" — оставили только значения до `5.0` (основная масса исходов краш-игр находится в этом диапазоне, значения выше — редкие крупные коэффициенты).
- Итоговый размер выборки после фильтрации: **42 997** значений.

#### 2. Построение гистограммы (интервальный вариационный ряд)
- Использовали **формулу Стерджесса** для определения оптимального числа интервалов:
  ```
  k = 1 + log₂(n) ≈ 17 интервалов
  ```
- Вычислили ширину интервала:
  ```
  h = (max - min) / k ≈ 0.235
  ```
- Для каждого интервала подсчитали частоту (`nᵢ`) и относительную частоту (`wᵢ = nᢵ / n`).

#### 3. Построение эмпирической функции распределения (ЭФР)
- Для каждого интервала вычислили накопленную относительную частоту.
- ЭФР показывает вероятность того, что коэффициент окажется меньше или равен заданному значению.

#### 4. Визуализация
- **Гистограмма распределения:** визуализирует, как часто выпадают те или иные коэффициенты.
- **График ЭФР:** ступенчатая функция, показывающая накопленную вероятность.

### 📌 **Вывод**

Построено эмпирическое распределение коэффициентов краш-игры на основе более 40 000 наблюдений. Распределение имеет ярко выраженную моду в области малых значений (1.1–1.3), что соответствует ожиданиям от данного типа игр. Полученные гистограмма и функция распределения могут служить основой для дальнейшего статистического анализа и проверки честности генератора случайных чисел (ГСЧ) игры.
